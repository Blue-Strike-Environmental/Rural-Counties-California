# Blue Strike Environmental, Dashboards

Interactive dashboards for Blue Strike Environmental's climate and GHG projects.

## Live URL

`https://dashboards.bluestrike.com` (pending DNS setup and Streamlit Cloud deploy).

## Projects

- **Rural Counties GHG Calculator** — Del Norte, Amador, Tuolumne. Baseline emissions and scenario modeling.

## Local development

```
pip install -r requirements.txt
streamlit run streamlit_app.py
```

Then open http://localhost:8501.

## Deployment

Deployed via Streamlit Community Cloud. Every push to `main` triggers an auto-redeploy.
